BlackHawk-Plugin-Device
----------

Device plugin of [BlackHawk](https://github.com/Lucky-Orange/BlackHawk).

##Install

No need, it's already built in BlackHawk.

##Use

```swift
// js code
console.log(device.model);
```

##Properties

* device.cordova
* device.model
* device.platform
* device.uuid
* device.version
